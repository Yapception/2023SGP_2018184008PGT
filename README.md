# 2023SGP_2018184008PGT
2023 Smartphone Game Programing using AndroidStudio

2023 스마트폰 프로그래밍 02반 2018184008 박기태

High Concept : 횡스크롤 점프 게임

핵심 메카닉 : 장애물과 바닥, 천장에 닿지 않게끔 점프를 통해 넘어가야함

  - 캐릭터와 구조물 충돌
